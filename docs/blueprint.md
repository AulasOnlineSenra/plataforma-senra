# **App Name**: Aulas Online Senra

## Core Features:

- Autenticação de Alunos: Login seguro via e-mail/senha ou Google OAuth com criação automática de usuários (incluindo os já existentes no WordPress) nome, e-mail e foto; e um painel administrativo para configurar credenciais Google.
- Seleção de Disciplinas: Permitir que os alunos escolham disciplinas, professores preferidos e o número de aulas diretamente na plataforma.
- Agendamento Automático: Habilitar o agendamento automático com confirmações imediatas para alunos e professores.
- Chat Integrado: Implementar um chat educacional integrado para comunicação contínua entre alunos e professores, assemelhando-se aos chats mais modernos e utilizados pelos usuários.
- Feedback da Sessão: Após cada aula, permitir que os alunos avaliem a sessão e forneçam feedback sobre o professor. Permitir que os administradores usem uma ferramenta para extrair palavras-chave positivas ou negativas frequentes para auxiliar a equipe de gestão na detecção de problemas, áreas de melhoria ou potenciais professores de destaque.
- Sistema de Pagamento: Permitir que os alunos comprem pacotes de aulas (por exemplo, 4, 8, 12 aulas) com integração PayPal e Mercado Pago.
- Painéis Baseados em Funções: Fornecer painéis abrangentes para administradores, alunos e professores, exibindo horários de aulas, histórico e status de pagamento. Garantir que os administradores possam visualizar todos os horários, enquanto os usuários veem apenas os seus. O administrador deverá poder ver todas as agendas, realizar novos agendamentos, cancelar agendamentos, realizar reagendamentos entre outras funcões administrativas
- Perfil do Professor: Local para o professor adicionar informações como disciplina, descrição, formação, idade, foto e disponibilidade de horários, assim quando os alunos quiserem agendar aulas terá apenas os horários que o professor selecionou para dar aula.
- Dashboard do Usuário: Cada usuário terá um dashboard no painel com histórico de aulas e agenda (com filtro diário, semanal e mensal).

## Style Guidelines:

- Cor primária: Azul claro (#A7D9ED) para evocar confiança e tranquilidade, complementando o ambiente educacional.
- Cor de fundo: Branco-sujo (#F5F4F4), proporcionando um ambiente limpo e sem distrações.
- Cor de destaque: Verde suave (#D0EFB1) para ações de confirmação e destaque de atualizações importantes, chamando a atenção de forma suave.
- Fonte do título: 'Belleza' sans-serif, para se alinhar com um visual de moda e arte.
- Fonte do corpo: 'Alegreya' serif para uma leitura confortável.
- Ícones consistentes e claros que representam diferentes funções e assuntos para garantir uma navegação fácil.
- Design limpo e intuitivo com uma hierarquia de informações clara, promovendo a usabilidade.
