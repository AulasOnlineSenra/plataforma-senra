# ai

Contém a lógica relacionada à inteligência artificial e integrações com agentes (MCP - Model Context Protocol).

## Subpastas ou arquivos típicos:
- Integrações com MCP servers (como Instagram DM, Meta Ads, Google Workspace)
- Lógica de agentes de IA para automação de marketing
- Funções para processamento de linguagem natural e tomada de decisão
- Configurações de conexão com serviços de IA externos

Este módulo é responsável por tornar a plataforma "inteligente", permitindo que ela execute tarefas como:
- Envio automático de mensagens diretas no Instagram
- Gerenciamento de campanhas de anúncios via Meta Ads
- Integração com Google Workspace para automação de tarefas
- Análise de dados e geração de insights
