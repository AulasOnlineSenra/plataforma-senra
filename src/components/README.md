# components

Contém componentes reutilizáveis de UI utilizados em toda a aplicação.

## Organização:
- Componentes de layout (como `app-sidebar.tsx`, `header.tsx`)
- Componentes de formulário reutilizáveis (inputs, textareas, etc.)
- Componentes específicos de funcionalidades (dashboard, financeiro, feedback, etc.)
- Pasta `ui`: componentes de interface de baixo nível (buttons, modals, etc.) - geralmente bibliotecas como shadcn/ui ou similares

Este módulo segue o padrão de componentes funcionais com TypeScript e Tailwind CSS para estilização.
